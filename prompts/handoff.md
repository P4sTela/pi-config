# Prepare an implementation handoff

Prepare a self-contained handoff for the next agent. If a next task was provided, include it: $@.

Inspect the current conversation context, git diff, relevant files, and validation state. Include:

1. Goal and current status
2. Decisions and constraints
3. Changed files and important symbols
4. Commands/tests already run and their results
5. Known risks or open questions
6. A precise next action

Do not modify project files and do not invent facts. Keep it concise but actionable.
